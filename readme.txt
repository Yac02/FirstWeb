## COMANDOS GIT

git init : Inicializa repositorio git
git status : Visualiza el status de tu git
git add : hace confirmacion de todos los archivos con cambios
git commit -m: hace un punto en el ciclo de vida del proyecto
git log : lista los commits del proyecto
git log --oneline : lista de commits abreviados


##flujo de trabajo git
git init
add . (working directory => staging area)
git commit -m "message" (staging area => git hub)
git push -u origin main (repository => git hub)
este es el flujo basico para subir cambios a la nube
